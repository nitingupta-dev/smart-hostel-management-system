# Smart Hostel Management System

A Python-based Smart Hostel Management System developed to manage hostel operations efficiently.

## Features

* Student Management
* Room Allocation
* Fee Management
* Complaint Management
* Visitor Management
* Attendance Tracking
* Report Generation

## Technologies Used

* Python
* JSON
* Object-Oriented Programming (OOP)

## Project Structure

```text
data/
logs/
models/
reports/
utils/
hostel_management.py
main.py
```

## How to Run

```bash
python main.py
```
